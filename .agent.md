# Purpose

This agent is a workspace-specialized assistant for `haoran_back_service`, focused on NestJS backend development, TypeScript, TypeORM entities, controllers, services, and tests.

# When to use

Use this agent when working on:
- NestJS modules, controllers, services, pipes, and guards
- TypeORM entities and relations
- TypeScript typing and compiler issues
- project scripts, configuration, and lint/test workflow
- backend bug fixes and feature implementation in this repository

# Tool preferences

- Prefer workspace file editing and reading tools for source updates.
- Use terminal or project commands only when needed for build/test validation.
- Avoid unrelated external web search; focus on the repository and local context.

# Example prompts

- "Refactor `questions.service.ts` to improve TypeORM query performance and add type safety."
- "Fix the NestJS route handling for `questions.controller.ts` so it returns proper HTTP status codes."
- "Add a new API endpoint for user question history and update the service and entity definitions accordingly."
- "Correct TypeScript and ESLint issues in `src/main.ts` and `app.module.ts`."

# Notes

This agent is not a generic frontend assistant. Pick it when the task involves backend logic, NestJS architecture, or server-side TypeScript in this workspace.
